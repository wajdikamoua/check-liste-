🔥 Configuration Firebase

Firebase est un service gratuit de Google. Une fois configuré, toutes les valeurs saisies seront partagées en temps réel sur tous les appareils.

1

Allez sur console.firebase.google.com → Créer un projet (nom au choix, désactiver Analytics).

2

Dans votre projet → Realtime Database → Créer une base de données → choisir un serveur → démarrer en mode test.

3

Paramètres du projet (⚙️) → Ajouter une application Web → copier l'objet firebaseConfig et coller ci-dessous.
